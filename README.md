## Tentang

**Invoi** adalah generator dokumen bisnis ringan untuk UMKM, freelancer, dan creator Indonesia. Buka, isi, custom tema, lalu unduh PDF atau gambar — data tersimpan lokal di browser, **gaperlu login**.

---

## Fitur

| | |
|---|---|
| 📄 **3 jenis dokumen** | Invoice, kwitansi, quotation |
| 🎨 **210+ tema** | Warna, font, dan layout siap pakai |
| 🖼️ **Background custom** | Upload sendiri atau 18 preset |
| ✍️ **Tipografi lengkap** | 36 font invoice + 24 font tanda tangan |
| 📱 **QRIS dinamis** | Nominal otomatis menyesuaikan total |
| ⬇️ **Export** | PDF & PNG high-res |
| 📦 **Inventori** | Simpan produk/jasa untuk isi cepat |
| 🔄 **Import** | Kompatibel JSON dari Invoiceku |
| 🌐 **Bilingual** | Indonesia & English |
| 📲 **Mobile-friendly** | Tab navigasi + preview responsif |

## Mulai Development

### Prasyarat

- [Node.js](https://nodejs.org/) 20+
- npm / pnpm / yarn

### Instalasi

```bash
git clone https://github.com/1tsprune/invoi.git
cd invoi
npm install
npm run dev
```

Buka **[http://localhost:3000](http://localhost:3000)**

Port custom:

```bash
npm run dev -- -p 3001
```

### Scripts

| Command | Keterangan |
|---------|------------|
| `npm run dev` | Jalankan dev server |
| `npm run build` | Build production |
| `npm run start` | Serve build production |
| `npm run lint` | Cek ESLint |

---

## Deploy

```bash
npm run build
npm run start
```

Rekomendasi deploy: **[Vercel](https://vercel.com)** — native support untuk Next.js.

Set environment variable di Vercel:

```bash
NEXT_PUBLIC_SITE_URL=https://invoi-nine.vercel.app
```

Ini untuk SEO (`metadataBase`, sitemap, OG image). Lihat `.env.example`.
